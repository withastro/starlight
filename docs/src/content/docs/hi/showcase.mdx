## प्लगइन्स

[प्लगइन्स](/hi/reference/plugins/) Starlight कॉन्फ़िगरेशन, UI और व्यवहार को अनुकूलित कर सकते हैं, और इनको साझा करना और पुन: उपयोग करना भी आसान है।
Starlight टीम द्वारा समर्थित आधिकारिक प्लगइन्स और Starlight उपयोगकर्ताओं द्वारा बनाए गए सामुदायिक प्लगइन्स के साथ अपनी साइट का विस्तार करें।
