Starlight sitenizi özel CSS dosyaları ile stilize edebilir veya Starlight Tailwind eklentisini kullanabilirsiniz.

## Özel CSS Stilleri

Starlight sitesine uygulanan stilleri özelleştirmek için, Starlight'ın varsayılan stillerini değiştirmek veya genişletmek için ek CSS dosyaları sağlayarak stil özelleştirebilirsiniz.

1. `src/` dizinine bir CSS dosyası ekleyin. 
	Örneğin, varsayılan sütun genişliğini artırabilir ve sayfa başlıkları için daha büyük metin boyutu ayarlayabilirsiniz:

   ```css
   /* src/styles/custom.css */
   :root {
   	--sl-content-width: 50rem;
   	--sl-text-5xl: 3.5rem;
   }
   ```

2. CSS dosyanızın yolunu `astro.config.mjs` içindeki Starlight'ın `customCss` dizisine ekleyin:

   ```diff lang="js"
   // astro.config.mjs
   import { defineConfig } from 'astro/config';
   import starlight from '@astrojs/starlight';

   export default defineConfig({
   	integrations: [
   		starlight({
   			title: 'Özelleştirilmiş CSS İle Dökümanlar',
   			customCss: [
   +				// Özel CSS dosyanızın göreli yolunu.
   +				'./src/styles/custom.css',
   			],
   		}),
   	],
   });
   ```

Starlight tarafından kullanılan ve sitenizi özelleştirmek için ayarlayabileceğiniz tüm CSS özel özellikleri, [GitHub'daki `props.css`](https://github.com/withastro/starlight/blob/main/packages/starlight/style/props.css) dosyasında görüntüleyebilirsiniz.

## Tailwind CSS

Astro projelerinde Tailwind CSS desteği, [Astro Tailwind entegrasyonu](https://docs.astro.build/en/guides/integrations-guide/tailwind/) ile sağlanır. Starlight, Starlight stilleriyle uyumlu olacak şekilde Tailwind'i yapılandırmaya yardımcı olan bir Tamamlayıcı Tailwind eklentisi sunar.

Starlight Tailwind eklentisi aşağıdaki yapılandırmayı uygular:

- Tailwind'in `dark:` varyantlarını Starlight'ın karanlık moduyla uyumlu hale getirir.
- Starlight'ın kullanıcı arayüzünde Tailwind [tema renklerini ve yazı tiplerini](#starlightı-tailwind-ile-stilize-etme) kullanır.
- Tailwind'in [Preflight](https://tailwindcss.com/docs/preflight) sıfırlama stillerini devre dışı bırakır ve yalnızca Tailwind'in kenar kullanım sınıfları için gerekli olan Preflight'in temel kısımlarını seçici bir şekilde geri yükler.

### Varolan bir projeye Tailwind eklemek

Eğer zaten bir Starlight sitesine sahipseniz ve Tailwind CSS eklemek istiyorsanız, aşağıdaki adımları takip edebilirsiniz.

1. Astro'nun Tailwind entegrasyonunu ekleyin:

    <Tabs>

    <TabItem label="npm">

    ```sh
    npx astro add tailwind
    ```

    </TabItem>

    <TabItem label="pnpm">

    ```sh
    pnpm astro add tailwind
    ```

    </TabItem>

    <TabItem label="Yarn">

    ```sh
    yarn astro add tailwind
    ```

    </TabItem>

    </Tabs>

2.  Starlight Tailwind eklentisini yükleyin:

    <Tabs>

    <TabItem label="npm">

    ```sh
    npm install @astrojs/starlight-tailwind
    ```

    </TabItem>

    <TabItem label="pnpm">

    ```sh
    pnpm install @astrojs/starlight-tailwind
    ```

    </TabItem>

    <TabItem label="Yarn">

    ```sh
    yarn add @astrojs/starlight-tailwind
    ```

    </TabItem>

    </Tabs>

3.  Tailwind'in temel stilleri için bir CSS dosyası oluşturun, örneğin `src/tailwind.css` adresinde:

    ```css
    /* src/tailwind.css */
    @tailwind base;
    @tailwind components;
    @tailwind utilities;
    ```

4.  Astro yapılandırma dosyanızı güncelleyerek, Tailwind temel stillerinizi kullanın ve varsayılan temel stilleri devre dışı bırakın:

    ```js {11-12,16-17}
    // astro.config.mjs
    import { defineConfig } from 'astro/config';
    import starlight from '@astrojs/starlight';
    import tailwind from '@astrojs/tailwind';

    export default defineConfig({
    	integrations: [
    		starlight({
    			title: 'Docs with Tailwind',
    			customCss: [
    				// Tailwind temel stillerinizin yolunu belirtin:
    				'./src/tailwind.css',
    			],
    		}),
    		tailwind({
    			// Varsayılan temel stilleri devre dışı bırakın:
    			applyBaseStyles: false,
    		}),
    	],
    });
    ```

5.  Starlight Tailwind eklentisini `tailwind.config.mjs` dosyasına ekleyin:

    ```js ins={2,7}
    // tailwind.config.mjs
    import starlightPlugin from '@astrojs/starlight-tailwind';

    /** @type {import('tailwindcss').Config} */
    export default {
    	content: ['./src/**/*.{astro,html,js,jsx,md,mdx,svelte,ts,tsx,vue}'],
    	plugins: [starlightPlugin()],
    };
    ```

### Starlight'ı Tailwind ile Stilize Etme

Starlight, kullanıcı arayüzünde [Tailwind tema yapılandırmanızdan](https://tailwindcss.com/docs/theme) değerler kullanacaktır.

Aşağıdaki seçenekler ayarlandıysa, Starlight'ın varsayılan stillerini geçersiz kılacaktır:

- `colors.accent` — bağlantılar ve geçerli öğe vurgusunda kullanılır
- `colors.gray` — arka plan renkleri ve sınırlar için kullanılır
- `fontFamily.sans` — UI ve içerik metni için kullanılır
- `fontFamily.mono` — kod örnekleri için kullanılır

```js {12,14,18,20}
// tailwind.config.mjs
import starlightPlugin from '@astrojs/starlight-tailwind';
import colors from 'tailwindcss/colors';

/** @type {import('tailwindcss').Config} */
export default {
	content: ['./src/**/*.{astro,html,js,jsx,md,mdx,svelte,ts,tsx,vue}'],
	theme: {
		extend: {
			colors: {
				// Tercih ettiğiniz vurgu renginiz. Starlight'ın varsayılanlarına en yakın olanı Indigo'dur.
				accent: colors.indigo,
				// Tercih ettiğiniz gri ölçeğiniz. Starlight'ın varsayılanlarına en yakın olanı Zinc'dir.
				gray: colors.zinc,
			},
			fontFamily: {
				// Tercih ettiğiniz metin yazı tipi. Starlight varsayılan olarak bir sistem yazı tipi yığını kullanır.
				sans: ['"Atkinson Hyperlegible"'],
				// Tercih ettiğiniz kod yazı tipi. Starlight varsayılan olarak sistem monospace yazı tipleri kullanır.
				mono: ['"IBM Plex Mono"'],
			},
		},
	},
	plugins: [starlightPlugin()],
};
```

## Tema

Starlight'ın renk teması, varsayılan özel özelliklerini geçersiz kılarak kontrol edilebilir.
Bu değişkenler, metin ve arka plan renkleri için kullanılan bir dizi gri tonu ve gezinmede bağlantıları ve geçerli öğeleri vurgulamak için kullanılan bir vurgu rengi ile arayüzün genelinde kullanılır.

### Renk tema düzenleyici

Starlight'ın vurgu ve gri renk paletlerini değiştirmek için aşağıdaki kaydırıcıları kullanın.
