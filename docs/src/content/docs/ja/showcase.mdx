## プラグイン

[プラグイン](/ja/reference/plugins/)により、Starlightの設定、UI、動作をカスタマイズできます。共有や再利用をおこなうことも簡単です。Starlightチームによってサポートされている公式プラグインと、Starlightユーザーによってメンテナンスされているコミュニティプラグインでサイトを拡張しましょう。
