Starlight 提供了一组内置图标，你可以使用 `<Icon>` 组件在内容中显示这些图标。

## 使用图标

可以使用 [`<Icon>`](/zh-cn/components/icons/) 组件来显示图标。
图标也常用于其他组件，例如 [卡片](/zh-cn/components/cards/) 或 [hero actions](/zh-cn/reference/frontmatter/#hero) 等设置。
