## 插件

[插件](/zh-cn/reference/plugins/) 可以自定义 Starlight 的配置、UI 和行为，同时也易于共享和复用。使用 Starlight 团队支持的官方插件和由 Starlight 用户维护的社区插件扩展你的网站。
