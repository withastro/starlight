## Hosting and Domain Configuration

- Subdomain managed in the TP Cloudflare Admin

## Managing Users

The WordPress admin has a section that manages adding new users and assigning the role/privelages. An admin user has the ability to add and remove users. The following roles are available:

- Admin - complete control of the WP Admin.
- Editor
- Author
